This is an example post created with the Hugo Tufte theme. The theme provides a beautiful minimalist design for blogs and academic publications.

The Hugo Tufte theme supports:

- Mathematical formulas via KaTeX
- Margin notes and sidenotes
- Beautiful code formatting
- Responsive design
